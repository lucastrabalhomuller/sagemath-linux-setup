# Installing Miniforge (Conda)

Miniforge provides a minimal Conda distribution using conda-forge,
which is the recommended source for scientific packages.

## Download

Visit:
https://conda-forge.org/download/

Download:
Miniforge3-Linux-x86_64.sh

## Installation

```bash
cd ~/Downloads
chmod +x Miniforge3-Linux-x86_64.sh
./Miniforge3-Linux-x86_64.sh
